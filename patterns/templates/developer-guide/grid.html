The grid is built on the Sassafras grid that comes with Sassaparilla. It aims to be be as flexible and as quick to implement as possible. As grid systems are fairly unique to each project, this page simply seeks to give you an overview of how the system is built.

</div>

Basic frame

The grid must be contained within a class of row.

<div class="row">
	<div class="colspan#-#"></div>
</div>

Adding column widths

To assign a width we need to use the 'colspan' synatax.

The first number represents the amount of grid columns available.

The second number represents the amount of columns you which to occupy.

<div class="row">
	<div class="colspan#-# as-grid"></div>
</div>

Floating columns

Adding the additional class of 'as-grid' floats the column to the left.

<div class="row">
	<div class="row__colspaced"></div>
</div>

Setting up gutters

If you need to have defined gutters between 'colspan#-#' you need to nest an additional wrapper with the class of 'row__colspaced'.

<div class="row">
	<div class="row__colspaced">
		<div class="colspan#-# as-grid with-gutter"></div>
	</div>
</div>

Combining gutters and floats

To apply gutters, you need to add an additional class of 'with-gutter' to the existing 'colspan#-# as-grid'.

<div class="row">
	<div class="colspan#-# as-grid with-leader leader-disable#"></div>
</div>

Adding and removing top vertical spacing

To apply top vertical spacing from the smallest breakpoint 'with-leader' to the existing 'colspan#-#'.

You can also apply vertical top spacing at any breakpoint by adding 'with-leader#'.

You can disable this spacing further up breakpoints by adding 'disable-leader#'.

<div class="row">
	<div class="colspan#-# as-grid with-trailer trailer-disable#"></div>
</div>

Adding and removing bottom vertical spacing

To apply top vertical spacing from the smallest breakpoint 'with-trailer' to the existing 'colspan#-#'.

You can also apply vertical top spacing at any breakpoint by adding 'with-trailer#'.

You can disable this spacing further up breakpoints by adding 'disable-trailer#'.
